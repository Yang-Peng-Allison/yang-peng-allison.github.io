Text mining + networks

Text Analytics of Australia's Energy Sector

An R text-mining pipeline over 23 documents from four source genres (Commentary, Industry, Media release, News), turning raw text into clustering, sentiment, and network insight.

  • Built a Document-Term Matrix with custom preprocessing (domain phrase merging, stemming, sparsity pruning) reduced to ~25 key tokens.
  • Cosine-distance + Ward.D hierarchical clustering of documents (k=4), evaluated by cluster purity (~52%).
  • Loughran-McDonald sentiment analysis by genre with Kruskal-Wallis and pairwise t-tests.
  • Three network visualisations built with ggraph: a single-mode document similarity network, a token co-occurrence network (fast-greedy communities), and a bipartite document-token network with edges coloured by document genre and tokens shaded by document frequency.
